
## Install

1. ```npm install```
2. ```npm install -g claudia```

## Configure

1. copy ```config-sample.js``` to ```config.js``` and edit it

## Read the docs

1. ```npm run info```

## Test locally

1. edit ```src/aws.js```
2. ```npm start```

## Deploy the first time

1. ```npm run create:aws```

## Update deployed api

- ```npm run publish:aws```
